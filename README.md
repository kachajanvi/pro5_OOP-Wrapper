💼 Employee Management System

📖 Overview

The Employee Management System is a Python console application developed using Object-Oriented Programming (OOP) principles. It provides a simple way to manage employee information through a menu-driven interface. The project demonstrates how inheritance and polymorphism can be used to organize different employee roles efficiently.

⸻

✨ Highlights

* 👤 Create Person Records
* 🆔 Add Employee Details
* 🏢 Manage Department Information
* 💻 Store Developer Programming Languages
* 📋 Display All Employee Records
* 🔍 Search Employees by Name
* 🚪 Simple Menu-Based Navigation
